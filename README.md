# gopher-glitch

A CLI tool to generate glitch art.

Original:

![original](https://i.imgur.com/kQm6qG3.jpg)

Gif:

![result](https://i.imgur.com/rMIq7sq.gif)

## Usage

```shell
$ gopher-glitch -filename venus.jpg -times 20
```

## Instalation

```shell
$ go install
```

## Config

Create a `config.json` file with your instructions.
